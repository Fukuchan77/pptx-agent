
Python プログラミング基礎講座

はじめに

本講座では、プログラミング初心者の方を対象に、Python プログラミングの基礎を学習します。Python は世界中で広く使用されているプログラミング言語で、シンプルな文法と強力な機能により、初心者から上級者まで幅広く支持されています。

この講座を通じて、変数、データ型、制御構文、関数などの基本的な概念を理解し、簡単なプログラムを作成できるようになることを目指します。実践的な例題と演習問題を通じて、プログラミングの基礎をしっかりと身につけていきましょう。

Python とは

プログラミング言語の概要

Python は 1991 年に Guido van Rossum によって開発されたプログラミング言語です。読みやすく書きやすいコードが特徴で、「コードの可読性」を重視した設計思想により、初心者にも学びやすい言語として知られています。

インデントによるブロック構造、豊富な標準ライブラリ、動的型付けなどの特徴により、短いコードで多くの処理を実現できます。Web 開発、データ分析、機械学習、自動化スクリプトなど、幅広い用途で活用されています。

オブジェクト指向、関数型、手続き型など、複数のプログラミングパラダイムをサポートしており、プロジェクトの要件に応じて適切なスタイルを選択できます。

Python の特徴

シンプルで読みやすい文法により、プログラミングの学習コストが低く抑えられています。英語に近い自然な記述が可能で、コードの意図が明確に伝わります。

豊富な標準ライブラリとサードパーティパッケージにより、様々な機能を簡単に実装できます。ファイル操作、ネットワーク通信、データベース接続など、一般的な処理は標準ライブラリで提供されています。

クロスプラットフォーム対応により、Windows、macOS、Linux など、異なる OS で同じコードを実行できます。開発環境の制約が少なく、チーム開発でも統一された環境を構築しやすい特徴があります。

Python の用途

Web アプリケーション開発では、Django や Flask などのフレームワークを使用して、効率的に Web サービスを構築できます。Instagram、YouTube、Dropbox など、大規模な Web サービスでも Python が採用されています。

データ分析と機械学習の分野では、NumPy、Pandas、scikit-learn、TensorFlow などのライブラリにより、高度な分析とモデル構築が可能です。データサイエンス分野で最も広く使用されている言語の一つです。

自動化スクリプトの作成により、日常的な作業を効率化できます。ファイル整理、レポート生成、システム管理など、繰り返し作業の自動化に最適です。

開発環境の準備

Python のインストール

公式サイト（python.org）から最新版の Python をダウンロードしてインストールします。Windows では実行ファイルをダウンロードして実行し、インストーラーの指示に従います。

macOS では Homebrew を使用したインストールが推奨されます。ターミナルで「brew install python3」コマンドを実行することで、最新版をインストールできます。

Linux では多くのディストリビューションで Python がプリインストールされていますが、最新版が必要な場合はパッケージマネージャーを使用してインストールします。

統合開発環境

Visual Studio Code は軽量で高機能なエディタで、Python 拡張機能により優れた開発体験を提供します。シンタックスハイライト、コード補完、デバッグ機能など、開発に必要な機能が揃っています。

PyCharm は Python 専用の統合開発環境で、プロフェッショナル向けの高度な機能を提供します。コード分析、リファクタリング、テスト実行など、大規模プロジェクトの開発に適しています。

Jupyter Notebook はインタラクティブな実行環境で、データ分析や機械学習の学習に最適です。コードとその実行結果、説明文書を一つのノートブックにまとめられます。

パッケージ管理

pip は Python の標準パッケージマネージャーで、サードパーティライブラリのインストールと管理を行います。「pip install パッケージ名」コマンドで簡単にライブラリをインストールできます。

仮想環境（venv）を使用することで、プロジェクトごとに独立したパッケージ環境を構築できます。異なるプロジェクト間でのパッケージバージョン衝突を防ぎ、環境を清潔に保てます。

requirements.txt ファイルにプロジェクトで使用するパッケージを記述することで、環境の再現性を確保できます。チーム開発やデプロイ時に同じ環境を簡単に構築できます。

基本的な文法

変数と代入

変数はデータを格納する箱のようなもので、名前を付けて値を保存できます。Python では明示的な型宣言が不要で、代入時に自動的に型が決定されます。

変数名は英字、数字、アンダースコアを使用でき、数字で始まることはできません。わかりやすく意味のある名前を付けることで、コードの可読性が向上します。

代入演算子「=」を使用して、変数に値を格納します。「x = 10」のように記述すると、変数 x に整数 10 が格納されます。複数の変数に同時に値を代入することも可能です。

データ型

整数型（int）は小数点を含まない数値を表現します。四則演算や比較演算など、様々な計算に使用できます。Python の整数は桁数の制限がなく、任意の大きさの整数を扱えます。

浮動小数点型（float）は小数点を含む数値を表現します。科学技術計算や統計処理で使用されますが、浮動小数点数の特性により、完全に正確な計算ができない場合があることに注意が必要です。

文字列型（str）は文字の列を表現し、シングルクォートまたはダブルクォートで囲んで記述します。文字列の連結、分割、検索など、豊富な操作メソッドが提供されています。

演算子

算術演算子には加算（+）、減算（-）、乗算（*）、除算（/）、剰余（%）、べき乗（**）があります。これらを組み合わせて複雑な計算を行えます。

比較演算子には等しい（==）、等しくない（!=）、大きい（>）、小さい（<）、以上（>=）、以下（<=）があります。条件判定で使用され、結果は真偽値（True/False）で返されます。

論理演算子には and（論理積）、or（論理和）、not（否定）があります。複数の条件を組み合わせた判定に使用され、複雑な条件式を構築できます。

制御構文

条件分岐（if 文）

if 文は条件に応じて処理を分岐させます。条件式が True の場合にインデントされたブロックが実行され、False の場合はスキップされます。

elif を使用して複数の条件を順番に評価できます。最初に True となった条件のブロックのみが実行され、それ以降の条件は評価されません。

else を使用してすべての条件が False の場合の処理を記述できます。if と elif のいずれの条件も満たさない場合に実行されるデフォルト処理として機能します。

繰り返し処理（for 文）

for 文はシーケンス（リスト、タプル、文字列など）の要素を順番に処理します。Python の for 文は「for 変数 in シーケンス」の形式で記述します。

range 関数と組み合わせることで、指定回数の繰り返し処理を実現できます。「for i in range(10)」は 0 から 9 まで 10 回繰り返します。

enumerate 関数を使用すると、インデックスと要素を同時に取得できます。リストの処理でインデックスが必要な場合に便利です。

繰り返し処理（while 文）

while 文は条件が True の間、処理を繰り返します。条件が False になるまで無限に繰り返されるため、ループを抜ける条件を適切に設定する必要があります。

break 文を使用してループを途中で抜けることができます。特定の条件が満たされた時点でループを終了したい場合に使用します。

continue 文を使用して現在の反復をスキップし、次の反復に進むことができます。特定の条件の場合のみ処理をスキップしたい場合に便利です。

データ構造

リスト

リストは複数の値を順序付きで格納するデータ構造です。角括弧で囲み、カンマで区切って要素を列挙します。「numbers = [1, 2, 3, 4, 5]」のように記述します。

インデックスを使用して特定の要素にアクセスできます。インデックスは 0 から始まり、「numbers[0]」で最初の要素を取得できます。負のインデックスを使用すると末尾から数えます。

append メソッドで要素を追加、remove メソッドで要素を削除できます。insert、extend、pop など、様々なメソッドによりリストを操作できます。

タプル

タプルはリストと似ていますが、一度作成すると要素を変更できない不変のデータ構造です。丸括弧で囲み、カンマで区切って要素を列挙します。

複数の値を一つの変数にまとめて返す場合や、変更されては困るデータを扱う場合にタプルを使用します。リストよりもメモリ効率が良く、ハッシュ可能なため辞書のキーとして使用できます。

タプルのアンパックにより、複数の変数に一度に値を代入できます。「x, y = (10, 20)」のように記述すると、x に 10、y に 20 が代入されます。

辞書

辞書はキーと値のペアを格納するデータ構造です。波括弧で囲み、「キー: 値」の形式で要素を記述します。「person = {"name": "太郎", "age": 25}」のように記述します。

キーを使用して対応する値にアクセスできます。「person["name"]」で "太郎" を取得できます。存在しないキーにアクセスするとエラーが発生するため、get メソッドの使用が推奨されます。

keys メソッドですべてのキー、values メソッドですべての値、items メソッドでキーと値のペアを取得できます。for 文と組み合わせて辞書の要素を順番に処理できます。

関数

関数の定義

関数は処理をまとめて名前を付けたもので、再利用可能なコードブロックです。def キーワードを使用して定義し、関数名と引数リストを指定します。

関数本体はインデントされたブロックで記述し、return 文で値を返します。return 文がない場合は None が返されます。

引数はカンマで区切って複数指定でき、デフォルト値を設定することも可能です。デフォルト値を持つ引数は省略可能になります。

関数の呼び出し

定義した関数は関数名と引数を指定して呼び出します。「result = add(5, 3)」のように記述すると、add 関数が実行され、戻り値が result に代入されます。

位置引数とキーワード引数の両方を使用できます。キーワード引数を使用すると、引数の順序を気にせず、わかりやすく関数を呼び出せます。

可変長引数を使用すると、任意の数の引数を受け取れます。「*args」で位置引数、「**kwargs」でキーワード引数を可変長で受け取れます。

スコープ

変数のスコープは、その変数が参照できる範囲を示します。関数内で定義された変数はローカル変数で、関数外からは参照できません。
